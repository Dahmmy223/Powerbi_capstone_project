# Powerbi_capstone_project
Analysis of FIFA WORLD CUP Five Year Tournament using Power 
## Project Overview

This project analyzed FIFA World Cup tournament performance from 2002 to 2018 using Microsoft Power BI.

## Objectives

• Compare team performance
• Analyse goals scored and conceded
• Identify the most successful countries
• Evaluate tournament trends

## Tools Used

- Power BI
- Power Query
- DAX
- GitHub

## Data Model

Star Schema

- Facts_Results
- Teams
- Years

## Key Insights

59 countries competed across the five tournaments 
640 games were played in total
793 goals were made
- Germany achieved the most goals over the Year
- Nigeria appeared 4 times in each tournament
- South Korea lost the most games.
